Email Authentication Record Generator

Build copy-paste DNS records for SPF and DMARC, plus a DKIM DNS template to help you deploy email authentication correctly.

Fix & Generate Records

Generates recommended templates for SPF + DMARC and a DKIM DNS template (DKIM keys must be generated by your provider/server).

SPF Builder

Authorized senders. Keep only one SPF record per domain.
Host/Name: @  |  Type: TXT
v=spf1 -all

DMARC Builder

Start with quarantine/reject when confident. Reports via rua.
Host/Name: _dmarc  |  Type: TXT
v=DMARC1; p=quarantine; fo=1; pct=100

DKIM DNS Template

We show DNS host + template value. Paste your provider’s public key into p=.
Host/Name: selector1._domainkey.example.com  |  Type: TXT
v=DKIM1; k=rsa; p=REPLACE_WITH_PUBLIC_KEY_FROM_PROVIDER
Tip: Get the key from Mailgun/Google Workspace/OpenDKIM/etc. This is a template.
Note: DKIM requires a provider-generated keypair. This page helps you build the correct DNS host and a valid template value.
